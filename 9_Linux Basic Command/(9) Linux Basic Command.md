    Macam2 Command linux
   Struktur commad linux
    Contoh
    ls = menampilkan semua file di penyimpanan
    namun tampa option dan parameter tidak berguna. 
    penggunaan command harus menggunakan option dan parameter
    
    Contoh basic command
    - cal = menampilkan kalender
    - date = menampilkan zona waktu di komputer
    
    Contoh File System Commands
    - touch = memperbolehkan membuat file baru yang kosong melalui baris perintah linux
    - cat = membuat daftar konten atau isi file.
    - cp = menyalin file ke file 1 ke file 2
    - mv = untuk memindahkan file.
    - rm = mengepush direktori beserta isinya. perlu hati hati karena sekali di hapus ga bisa di balikin
    - mkdir = membuat direktori baru
    - rmdir = menghapus direktori
    - cd = menjelajahi atau pindah diroktori
    - pwd = untuk mengetahui lokasi yang di gunakan
    
    contoh Processes control commands
    - top = menampilkan daftar proses yang sedang berlangsung
    - clear = untuk membersihkan terminal
    - history = untuk melihat commad yang sudah pernah di gunakan

    contoh utilities programs commands
    - ls = menampilkan semua file di directori kita
    - which = command yang digunakan untuk mengeksekusi file yang di berikan
    - sudo = menjalankan hak akses admin
    - find = untuk mencari file dan directory

    contoh file access permissions
    - chmod = untuk membaca, menulis, dan menjalankan permision
    - chown = mengubah atau menstransfer file ke user pemilik
